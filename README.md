# Xidong CCTV Live Map

Static CCTV map page for GitHub Pages.

## Folder structure

```text
xidong-cctv/
├── index.html
├── README.md
├── .gitignore
└── assets/
    └── cctv-map.png
```

## Update the map image

Replace:

```text
assets/cctv-map.png
```

with a new image using the same file name, then commit the change.
